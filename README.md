#  satirical-ai-detector

joke web app that pretends to be a sophisticated ai detector made to prank my little sister The application accepts **any** user-uploaded image. After a fake "forensic" scanning animation, it will *always* return a static, predefined report claiming the image is "100% AI Generated" with a randomized confidence score and a random list of "key artifacts."

## Tech Stack

This project is built with 100% client-side code:
* **HTML**
* **CSS** (with CSS Custom Properties for theming)
* **JavaScript** (vanilla JS, no frameworks)

## How to Run

No build step or server is required.

1.  Clone this repository.
2.  Open the `index.html` file directly in your web browser.
